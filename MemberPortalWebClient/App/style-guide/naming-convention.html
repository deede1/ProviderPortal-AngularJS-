File Naming

  • Use a dash for spacing. eg (HelloWorld.html vs Hello-World.html)
  • Lower-case
  • Three part names for angular files.
    1. Name-of-the-thing
    2. type-of-the-thing (eg. .controller, .directive, .service)
    3. file-extension (eg. .js, .html)

See Directory Structure for more examples.

                     my-cool-thing.directive.html
                    my-cool-thing.directive.js
                    some-other-cool-thing.service.js
                    this-page.controller.js

URL Route Naming

  • Use a dash for spacing. eg (/my-section/hello-world vs /mysection/helloworld)
  • Define Route/State URL's in lowercase only.
    • We have configured the $urlProvider to accept mixed-case url input by normalizing requested url's to lower-case. but if a state's url is defined with mix-casing the normalized value will not match.
